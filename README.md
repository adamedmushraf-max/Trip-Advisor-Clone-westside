TripAdvisor Clone

A fully responsive TripAdvisor-inspired travel website built using HTML5 and CSS3. This project replicates the look and feel of the TripAdvisor homepage, featuring a modern user interface, travel destination showcases, recommendation sections, and a clean navigation experience.

The project was developed to strengthen my front-end development skills and gain hands-on experience in creating responsive layouts, organizing content effectively, and implementing modern web design principles using only HTML and CSS.

✨ Features
Responsive navigation bar
Hero section with travel search interface
Popular destinations showcase
Travel inspiration and recommendation sections
Interactive hover effects
Clean and modern UI design
Mobile-friendly responsive layout
Organized and reusable code structure
